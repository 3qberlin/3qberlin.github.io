# color
## 以main取代primary文字
## 以小駝峰式命名

  主色：main
  主色深：mainDark
  主色淺：mainLight
  主色透明：mainTransparent

  次主色：secondary
  次主色深：secondaryDark
  次主色淺：secondaryLight

  配色橘：orange
  配色膚色：skin
  配色藍：blueLight

  灰：深到淺 → 1 ~ 6
  gray1
  gray2
  gray3
  gray4
  gray5
  gray6
  
 
  Example:
  bg-mainDark , text-grayDark, border-orange
